PEMBAHASANPresent Continuous

Tenses ini berfungsi yang digunakan untuk menjelaskan Kegiatan yang terjadi nya sekarang atau Kegiatan yang sedang berlangsung saat ini.

Ciri Present Continuous

=> Now

=> Right now

=> today

=> at the moment ✅

Rumus Present Continuous(+) S + Be (is/am/are) + V-ing(-) S + Be (is/am/are) + not + V-ing(?) Be (is/am/are) + S + V-ing?

Catatan

➡️Subjek + Be

I + amHe/she/it/satu nama + isYou/we/they/dua nama + are

KESIMPULAN

11. -I had waiting for the package when she came. - I hadn't waiting for the package when she came. - Had you waiting for the package when your brother came?bantu buatin pattern nya bang​


The correct grammar for these sentences would be:

•I was waiting for the package when she came. (Past continuous tense)

•I hadn't been waiting for the package when she came. (Past perfect continuous tense)

•Had you been waiting for the package when your brother came? (Past perfect continuous tense, question form)

The pattern for each sentence would be:

Subject + past continuous tense (was/were + present participle) + object + when + past simple tense (subject + came)

Subject + hadn't + past participle of "be" (been) + present participle + object + when + past simple tense (subject + came)

Had + subject + been + present participle + object + when + past simple tense (subject + came) + question mark

Jawaban:

Kalimat 1:

Subject (I) + past perfect (had been waiting) + object (for the package) + when clause (she came)

Kalimat 2:

Subject (I) + negative past perfect (hadn't been waiting) + object (for the package) + when clause (she came)

Kalimat 3:

Auxiliary (Had) + subject (you) + past participle (been waiting) + object (for the package) + when clause (your brother came)

Angkatan / generasi yang lebih tua dari generasi sekarang sadar bahwa angkatan mereka tuh banyak melakukan kesalahan (perang, etc)
Melalui lirik-liriknya, white lion mencoba menjelaskan supaya kita, angkatan / generasi berikutnya bisa mengerti dan memaafkan kesalahan mereka.
