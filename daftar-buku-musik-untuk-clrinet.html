25. Pada suatu rak buku terdapat 2 buku fisika, 2buku kimia, 5buku biologi. Disusun berdampingan maka banyak cara untuk menyusun buku trsbt adalah.....


Saya coba bantu

pertama, anggap semua buku itu satu bagian, jadi ada 3 bagian penyusunan, sehingga terapat 3 x 2 x 1 = 6 cara

selanjutnya :
buku fisika = 2 cara penyusunan
buku kimia = 2 cara penyusunan
buku biologi = 5! = 5 x 4 x 3 x 2 x 1 = 120 cara penyusunan

Jadi total cara penyusunan adalah 6 x 2 x 2 x 120 = 2880 cara

28. Langkah untuk membuat daftar isi di readium buku digital adalah


1. Siapkan file word berekstensi .doc atau docx. Kalau bisa, gunakan sebuah dokumen penuh teks dengan teks yang telah diatur Heading-nya. Jika ada gambar-gambar, atur gambar-gambar tersebut In Line With Text di Ms Word. Jika ada table, atur table tersebut dengan lebar 100% merata dengan teks.

2. Kemudian Save dokumen tersebut sebagai HTML.

3. Buka program Sigil. Jika belum terinstall, kita dapat mendownloadnya di sini.

4. Setelah masuk ke software Sigil, buat sebuah dokumen epub baru

5. Tambahkan sebuah file html

yang telah kita buat sebelumnya. Dengan klik Add Existing File, Open file html tersebut.

6. Berikut tampilan dokumen setelah file html telah dibuka. Di sini kita dapat mulai mengatur Table of Content dan Meta Data. Table of Content di sini adalah daftar isi untuk dokumen ePub, bukan untuk dokumen Ms Word.

7. Buat Table of Content dengan meng-klik Tools > Table of Contents > Generate Table of Contents.

8. Kemudian kita akan mencoba mengatur Metadata dokumen ePub agar dokumen ePub memiliki informasi yang sesuai saat terpublish. Caranya, Klik Tools > Metadata Editor. Kemudian atur Judul dan Pengarang.

9. Dokumen ePub juga dapat diberi Cover, kita dapat membuat Cover dengan klik Tools > Add Cover. Kemudian pilih file image yang akan dijadikan Cover. Klik OK.

Kemudian kita akan mencoba mengatur Metadata dokumen ePub agar dokumen ePub memiliki informasi yang sesuai saat terpublish. Caranya, Klik Tools > Metadata Editor. Kemudian atur Judul dan Pengarang.

9. Dokumen ePub juga dapat diberi Cover, kita dapat membuat Cover dengan klik Tools > Add Cover. Kemudian pilih file image yang akan dijadikan Cover. Klik OK.

10. Kita juga dapat menyisipkan Video ke dalam dokumen ePub yang dapat dimainkan di Readium. Caranya, klik Insert > File. Kemudian pilih video yang akan disisipkan. Klik Ok.
