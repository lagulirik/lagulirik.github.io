1. Algoritma yang mencari informasi berdasarkan kata kunci mulai dari awal array hingga akhir array adalah


Jawaban:

Salah satu algoritma yang dapat digunakan untuk mencari informasi berdasarkan kata kunci dari awal array hingga akhir array adalah algoritma Linear Search. Berikut adalah pseudocode untuk algoritma Linear Search:

```

1. Inisialisasi posisiAwal = -1 (berarti kata kunci tidak ditemukan)

2. Untuk setiap elemen arr[i] dalam array:

a. Jika arr[i] sama dengan kata kunci:

i. Set posisiAwal = i

ii. Hentikan pencarian

3. Kembalikan posisiAwal

```

Dalam algoritma ini, pencarian dimulai dari awal array dan berlanjut hingga akhir array. Jika kata kunci ditemukan, posisi awal diperbarui dengan indeks di mana kata kunci ditemukan. Jika kata kunci tidak ditemukan, posisiAwal tetap -1.

Berikut adalah contoh implementasi algoritma Linear Search dalam bahasa Python:

```python

def linear_search(arr, kata_kunci):

posisi_awal = -1

for i in range(len(arr)):

if arr[i] == kata_kunci:

posisi_awal = i

break

return posisi_awal

```

Anda dapat memanggil fungsi `linear_search` dengan memberikan array dan kata kunci yang ingin dicari sebagai argumen. Fungsi akan mengembalikan indeks pertama di mana kata kunci ditemukan, atau -1 jika kata kunci tidak ada dalam array.

Contoh penggunaan:

```python

array = [4, 2, 7, 1, 9, 5]

kata_kunci = 7

posisi = linear_search(array, kata_kunci)

print("Kata kunci ditemukan pada indeks:", posisi)

```

Output:

```

Kata kunci ditemukan pada indeks: 2

```

Dalam contoh di atas, kata kunci "7" ditemukan pada indeks ke-2 dalam array.
