22. Tulislah nama senyawa berikut! a. HIO b.HIO2 c. HIO3 d.HIO4


Nama senyawa dari :

a. HIO = asam hipoiodit

b. HIO₂ = asam iodit

c. HIO₃ = asam iodat

d. HIO₄ = asam periodat

Pembahasan

Tata Nama Senyawa

Untuk mengetahui nama suatu senyawa terlebih dahulu kita harus memahami tentang lambang unsur dan nama unsur kimia. Hal tersebut merupakan ilmu dasar yang harus diketahui untuk memperdalam tentang tata nama senyawa. Penamaan senyawa diatur oleh IUPAC (International Union Pure and Applied Chemistry). berdasarkan kesepakatan oleh para ahli ilmuan didunia.

A. Tata Nama Senyawa Anorganik

1. Senyawa Biner dari Logam dan Non Logam

a. Logam yang memiliki bilangan oksidasi 1 jenis (Golongan IA, IIA, IIIA)

1) Menyebutkan nama logam nya

2) Menyebutkan nama non logam nya + akhiran –ida

Contoh:

NaCl = natrium klorida

MgF2 = magnesium florida

CaCl2 = kalsium klorida

Na2O = natrium oksida

b. Logam yang memiliki bilangan oksidasi lebih dari 1 jenis (logam transisi)

1) Menyebutkan nama logam nya diikuti angka romawi sesuai dengan bilangan oksidasi unsur logam

2) Menyebutkan nama non logam nya + akhiran –ida

Contoh :

FeO = besi (II) oksida

Cu2O = tembaga (II) oksida

PbI2 = timbal (II) iodida


2. Senyawa Biner Non Logam dan Non Logam

1) Menyebutkan nama Yunani jumlah atom non logam ke-1 diikuti dg nama unsur non logam ke-1

2) Menyebutkan nama Yunani jumlah atom non logam ke-2 diikuti dg nama unsur non logam ke-2 + akhiran –ida

Jumlah Atom dalam Bahasa Yunani

2 Di

29. Tentukan bilangan oksidasi I dalam senyawa senyawa berikut! A. HIO : asam hipoiodit B. HIO2 : asam iodit C. HIO3 : asam iodat D. HIO4 : asam periodat


Bilangan oksidasi atom I dalam senyawa HIO adalah +1. Bilangan oksidasi atom I dalam senyawa HIO₂ adalah +3. Bilangan oksidasi atom I dalam senyawa HIO₃ adalah +5. Bilangan oksidasi atom I dalam senyawa HIO₄ adalah +7.

Pembahasan

∴ Bilangan oksidasi I pada HIO adalah +1.

Untuk memperoleh bilangan oksidasi ( biloks) atom I dapat dilakukan dengan sutitusi dari bilangan oksidasi yang sudah diketahui pada senyawa HIO

∵Biloks total HIO adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 ) dan bilangan oksidasi  H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

HIO= 0

(jumlah atom H x biloks atom H) + (jumlah atom I x biloks atom I) + (jumlah atom O x biloks atom O) = 0

(1 x 1) + (1 x biloks I) + (1x (-2)) = 0

1 + Biloks I - 2 = 0

Biloks I= 2-1

Biloks I = +1

∴ Bilangan oksidasi I pada HIO₂ adalah +3.

Untuk memperoleh bilangan oksidasi ( biloks) atom I dapat dilakukan dengan sutitusi dari bilangan oksidasi yang sudah diketahui pada senyawa HIO₂

∵Biloks total HIO₂ adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 ) dan bilangan oksidasi  H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

HIO₂= 0

(jumlah atom H x biloks atom H) + (jumlah atom I x biloks atom I) + (jumlah atom O x biloks atom O) = 0

(1 x 1) + (1 x biloks I) + (2x (-2)) = 0

1 + Biloks I - 4 = 0

Biloks I= 4-1

Biloks I = +3

∴ Bilangan oksidasi I pada HIO₃ adalah +5.

Untuk memperoleh bilangan oksidasi ( biloks) atom I dapat dilakukan dengan sutitusi dari bilangan oksidasi yang sudah diketahui pada senyawa HIO₃

∵Biloks total HIO₃ adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 ) dan bilangan oksidasi  H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

HIO₃= 0

(1 x 1) + (1 x biloks I) + (3x (-2)) = 0

1 + Biloks I - 6 = 0

Biloks I= 6-1

Biloks I = +5

∴ Bilangan oksidasi I pada HIO₄ adalah +7.

Biloks I= 8-1

[H+] = √(Ka/Kb) x Kw
= √ (2,3x10^-11/1,8x10^-5) x 10^-14
